# Better Perplexity AI 🔍🤖

A full-stack AI-powered question-answering web app that generates answers using AI and provides sources, with support for follow-up queries.

---

## 🚀 Features

- Ask any question and get AI-generated answers  
- Displays relevant sources alongside answers  
- Supports follow-up questions (conversation context)  
- Clean and minimal UI  
- Real-time responses  

---

## 🛠️ Tech Stack

Frontend:
- React
- Vite
- TypeScript

Backend:
- Node.js
- Express
- TypeScript

APIs:
- OpenAI API

Tools:
- Hoppscotch (API testing)
- Git & GitHub

---

## ⚙️ Setup Instructions

1. Clone the Repository

git clone https://github.com/GurajalaThristha/better-perplexity-.git
cd better-perplexity-

2. Install Dependencies

Frontend:

cd client
npm install
npm run dev

Backend:

cd server
npm install
npm run dev

3. Environment Variables

Create a .env file inside the server/ folder and add:

OPENAI_API_KEY=your_openai_api_key
PORT=5000

---

## ▶️ How to Use

1. Start backend server  
2. Start frontend  
3. Open browser → http://localhost:5173  
4. Enter a question  
5. View answer + sources  
6. Ask follow-up questions like:
   - Explain simply  
   - Give example  
   - Short answer
